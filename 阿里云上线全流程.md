# 天成测绘网站 — 阿里云 OSS 静态托管上线全流程

> 目标：自有域名 + 阿里云 OSS + CDN + 百度收录
> 费用：~10 元/月

---

## 一、买域名 + ICP 备案（7-20 天，最先做）

### 1.1 阿里云买域名
1. 打开 [wanwang.aliyun.com](https://wanwang.aliyun.com)
2. 搜索 `sztcch.com.cn`（你名片上印的）或 `tiancheng-ch.com`
3. 选 `.com.cn` / `.cn` / `.com`，加入购物车，付款

### 1.2 ICP 备案
1. 阿里云控制台 → **ICP 备案** → 开始备案
2. 准备：身份证正反面、手持身份证照片、营业执照（如有）
3. 填写网站信息，服务商选「阿里云」
4. 阿里云初审（1-2 天）→ 管局审核（7-15 天）
5. 通过后会收到**备案号**（如粤ICP备XXXXXXXX号）

> ⚠️ 备案期间网站要关停，不能访问。等备案过了再做后面步骤。

---

## 二、OSS 存储桶 + CDN 配置（备案通过后，30 分钟）

### 2.1 创建 OSS 存储桶
1. 阿里云控制台 → **对象存储 OSS** → 创建 Bucket
2. 名称：`tiancheng-survey`（或自定义）
3. 区域：华南（深圳），离公司近
4. 读写权限：**公共读**
5. 版本控制：不开

### 2.2 开启静态网站托管
1. Bucket → 数据管理 → **静态页面**
2. 默认首页：`index.html`
3. 默认 404 页：`index.html`（React SPA 需要，让前端路由接管）

### 2.3 配置 CDN 加速
1. Bucket → **传输加速** → 开启 CDN 加速
2. CDN 域名：`tiancheng-survey.your-domain.com` 或直接用主域名
3. 加速区域：**仅中国大陆**
4. 源站类型：OSS 域名

### 2.4 域名 CNAME
1. CDN 控制台 → 域名管理 → 复制 CNAME 地址（形如 `xxx.aliyuncs.com`）
2. 阿里云 DNS 解析 → 添加记录：
   - 记录类型：CNAME
   - 主机记录：`@`（主域名）或 `www`
   - 记录值：粘贴 CNAME 地址
3. 等待 DNS 生效（几分钟到几小时）

---

## 三、上传网站文件（3 分钟）

### 3.1 构建生产版本
```bash
cd tiancheng-survey
npm run build
```
生成 `dist/` 目录。

### 3.2 上传到 OSS
1. 阿里云 OSS 控制台 → Bucket → 文件管理
2. 把 `dist/` 里面的所有文件拖进去（不是 dist 文件夹，是里面的东西）
3. 或用命令行上传：
```bash
# 安装 OSS CLI
npm install -g ossutil
ossutil config  # 按提示输入 AccessKey
ossutil cp -r dist/ oss://tiancheng-survey/ --update
```

### 3.3 设置缓存策略
- `.html` 文件：缓存 0 秒（每次更新立即生效）
- `.js / .css / .jpg / .png` 文件：缓存 30 天（带 hash 不会冲突）
- 在 CDN 控制台 → 缓存配置 → 添加规则

## 四、域名上线（备案通过后）

### 4.1 修改前端配置
```diff
- vite.config.ts: base: "/tiancheng-survey/"
+ vite.config.ts: base: "/"

- src/routes/index.tsx: basename: "/tiancheng-survey"
+ src/routes/index.tsx: 删除 basename 这一行
```
改完重新 `npm run build`，重新上传。

### 4.2 验证
```
浏览器输入你的域名 → 应该打开网站首页
```

## 五、百度收录（上线后，半小时）

### 5.1 添加站点
1. 打开 [ziyuan.baidu.com](https://ziyuan.baidu.com)（百度站长平台）
2. 添加站点 → 输入 `https://你的域名`
3. 验证身份（选「文件验证」→ 下载 HTML → 上传到 OSS 根目录 → 点验证）

### 5.2 提交 sitemap
1. 生成 sitemap.xml（放 `public/sitemap.xml`，Vite 构建会输出到 dist）
2. 百度站长 → **数据引入** → 链接提交 → 提交 sitemap

### 5.3 主动推送（加速收录）
每次更新网站，跑这个命令：
```bash
curl -H 'Content-Type:text/plain' --data-binary @sitemap.txt "http://data.zz.baidu.com/urls?site=你的域名&token=你的token"
```

## 六、配置 HTTPS（必须）

阿里云 CDN → **域名管理** → **HTTPS 配置**
1. 开启 HTTPS
2. 申请免费证书（阿里云提供 DigiCert 免费版，一年有效）
3. 强制 HTTPS 跳转：开启

---

## 七、表单接收咨询（接 Formspree）

修改 `ContactForm.tsx` 表单的 `action` 属性：
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```
注册 [formspree.io](https://formspree.io)，创建一个 Form，拿到 ID，填入即可。

有人提交咨询 → 自动发邮件到 `szchzxx@126.com`。
